Penguin Guano Classification API Demo

How to use the Penguin Colony Segmentation API with Google Earth Imagery

Follow the steps below:

Step 1
Download Google Earth or Google Earth Pro here. Google Earth Pro is now free, and contains additional features that users may find useful.
Step 2
Download the .kml files for all Adélie penguin sites at here. Double click the file to open in Google Earth when downloaded. When opened they will help you identify existing colonies.
Step 3
Using the .KML files you downloaded in step 2, explore some of the known penguin colonies in Google Earth. This will help orient you to what a colony should look like! You can see some images here. If you can't find the colony, try using the time slider to find a clearer image. Bigger colonies will be much easier to spot than smaller ones.
Keep in mind that when we look for penguins from satellite, we aren't looking for individual penguins. We actually look for guano stains.
Step 4
Extract the images of penguin colonies from Google Earth Pro. Once we localize the penguin colony, keep the eye altitude at ~1 km and save the image at 1140x802 resolution with the “.jpg” extension.

See the below example image of “Cape Crozier” colony.

Step 5
Use the API to segment the colony from the image using the “upload” tab.
